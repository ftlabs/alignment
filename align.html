Align

A (very) experimental look at recent articles in FT.com, exploring the visual impact of aligning the search results in novel ways.
The 'key text' is used to identify articles (via keyword search) and to align the results.
It is case sensitive, and the whole thing breaks if there are no matching search results, so choose wisely.

key text:
any
title-only

The code for all this sits at github.com/railsagainstignorance/alignment
The code has very little error handling, and it can not work for an excitingly wide variety of reasons.
Perhaps the main reason is that this is MyFirstGoProgram.
